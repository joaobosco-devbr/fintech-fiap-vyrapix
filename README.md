# VyraPix - Dashboard Financeiro

Projeto organizado para entrega da atividade Fintech.

## Estrutura

```text
vyrapix-projeto-organizado/
├── index.html
├── css/
│   ├── tailwind.css
│   └── style.css
└── img/
    ├── logo.jpeg
    └── logo-fundo.png
```

## Como abrir

Abra o arquivo `index.html` no navegador ou use a extensão Live Server no VS Code.

## Observações

- `css/tailwind.css`: arquivo base com as classes utilitárias do projeto.
- `css/style.css`: estilos personalizados, como logo, marca d'água e ajustes de responsividade.
- `img/logo.jpeg`: logo principal da sidebar.
- `img/logo-fundo.png`: logo usada como marca d'água no dashboard.
